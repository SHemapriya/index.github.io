ACHIEVEMENT

WEEKLY SANTHE

Description:

The project involves developing a user-friendly system with a focus on UI/UX,databases, APIs, and Android testing. It ensures data security through login and password methods and serves two user groups: Framer and Merchant, offering features like negotiation, payments, feedback, and a virtual marketplace. This real-time system earned 3rd place in a national-level hackathon organized by DLithe Company, highlighting its innovation and practicality.
